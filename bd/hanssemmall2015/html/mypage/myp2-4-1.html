결제수단 변경

결제수단 신용카드
  • 신용카드, 실시간 계좌이체는 안전거래 서비스 LGU+를 통해 결제 됩니다.
  • Paynow(간편결제)는 어플리케이션 설치후 자주 사용하는 결제수단을 최초1회 등록하면, 카드, 지갑 없이도 바로 결제 할 수 있는 결제 서비스입니다. (PC에서 주문시 간편결제>핸드폰수단이 제공되지 않습니다.)

안심클릭서비스

안심클릭 서비스란?

전자상거래 신용카드 결제 시, 카드 소지 고객이 사전에 등록해 둔안심클릭 비밀번호를 입력하여 타인의 불법 카드 도용 사고를 방지하는 편리하고 안전한 신개념 전자상거래 인증 서비스입니다.

대상카드 : LG / 삼성 / 외환 / 현대 / 신한 / 하나 / 한미 / 롯데카드

안심클릭 결제방법

카드번호와 사전 등록된 안심클릭 비밀번호(혹은 공인인증서)의 입력만으로 결제가 완료됩니다. LG/삼성/외환/현대/신한/하나/한미/롯데카드 중 소유하고 계신 카드사의 홈페이지에서 간단히 신청하실 수 있습니다.
  • 01 - 결제페이지 결제수단 선택 : KB/BC카드
  • 02 - 안전 결제(ISP)인증 팝업 : 비밀번호 입력
  • 03 - 결제완료

공인인증서 사용여부

금액에 상관없이 안심클릭과 공인인증서 중에서 선택 결제가 가능합니다.

안전결제(ISP)서비스안내

안전결제(ISP) 란?

스토어 이용 시 고객이 카드번호/비밀번호 등을 입력함으로써 발생될 수 있는 개인정보 유출 및 카드도용의 문제점을 차단하기 위해 안전결제(ISP) 비밀번호만으로 안전한 전자상거래를 보장하는 서비스입니다.

대상카드 : KB(국민) / BC / 우리(구 평화) / 조흥(구 강원)

안전결제(ISP) 결제방법

고객님이 사전에 등록한 ISP 비밀번호만을 입력만으로 결제가 완료됩니다.KB(국민)/BC/우리(구 평화)/조흥(구 강원)카드 중 소유하고 계신 카드사의 홈페이지에서 간단히 신청하실 수 있습니다.
  • 01 - 결제페이지 결제수단 선택 : KB/BC카드
  • 02 - 안전 결제(ISP)인증 팝업 : 비밀번호 입력
  • 03 - 결제완료

공인인증서 사용여부

30만원 이상 거래시는 안전결제(ISP)+공인인증 확인의 2중 보호장치로 더욱 안전합니다.

공인인증서 발급 방법 안내

공인인증서란?

일종의 '사이버인감증명서'로 고객이 사전 설정해둔 별도 비밀번호를 입력하여 전자상거래 사고를 방지하는 수단입니다.　KB/BC 카드의 경우는 30만원 이상 결제 시 반드시 공인인증서를 의무사용하나 나머지 카드의 경우는 안심클릭과 공인인증 중 선택하여 사용 가능합니다.

안심클릭 결제방법

공인인증서는 본인 확인 절차 상, 최초 1회는 은행 등 공인인증기관을 직접 방문하여 발급 받아야 합니다.
  • 01 - 결제페이지 결제수단 선택 : KB/BC카드
  • 02 - 안전 결제(ISP)인증 팝업 : 비밀번호 입력
  • 03 - 결제완료

공인인증서 사용여부

금액에 상관없이 안심클릭과 공인인증서 중에서 선택 결제가 가능합니다.
